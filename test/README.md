# Code Clash: Tactical Integration Package (Level 1/1)

This folder contains the standalone **Code Clash** tactical component, identical to the `game/play/1/1` interface. It is designed for easy integration into other React/Next.js projects.

## Contents
- `CodeClashTactical.js`: The main game component. It is **self-contained**, procedurally generating all its own textures and assets (no external images required).
- `page.js`: A Next.js page wrapper to preview the component at `/test`.

## How to Integrate
1. **Install Phaser**:
   ```bash
   npm install phaser
   ```
2. **Copy the Component**:
   Copy `CodeClashTactical.js` into your project's `components` (or equivalent) directory.
3. **Usage**:
   ```javascript
   import CodeClashTactical from './CodeClashTactical';

   export default function MyGamePage() {
     return (
       <div className="game-container">
         <CodeClashTactical />
       </div>
     );
   }
   ```

## Key Features
- **Tactical Cockpit HUD**: Full health monitor, mission objectives, compass, and weapon arsenal.
- **Hunting Loop**: Defeat enemies (Bugs) to drop "Logic Treasures" which teach programming concepts.
- **Zero-Asset Loading**: All visuals (buildings, characters, grass, icons) are drawn on the fly using the Phaser Graphics API.
